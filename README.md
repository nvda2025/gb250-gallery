# GB250 Clubman Touring Gallery

Honda GB250 Clubman の写真と動画を、ヴィンテージ雑誌のような雰囲気で楽しむためのサイトです。

## 起動方法

1. このフォルダ全体をローカルPCの任意の場所に保存します。
2. `index.html` をダブルクリック（またはブラウザにドラッグ＆ドロップ）して開いてください。
   - ※サーバーを立てる必要はありません。
   - ※インターネット環境がない場所でも、フォント以外は正常に表示されます。

## ファイルの配置方法

サイトを正常に表示させるために、お手持ちのファイルを以下のフォルダに配置してください。

- **画像ファイル**: `images/` フォルダの中に配置
  - `IMG_1310.JPG` 〜 `IMG_2802.JPG` （全16枚）
- **動画ファイル**: `videos/` フォルダの中に配置
  - `動画.MOV`

## カスタマイズ方法

### 写真の追加
1. `images/` フォルダに新しい画像を追加します。
2. `index.html` の `<div class="gallery-container">` 内に以下のコードをコピー＆ペーストして追加します。
   ```html
   <div class="gallery-item reveal">
       <img src="images/ファイル名.JPG" alt="説明" loading="lazy">
       <div class="gallery-overlay">
           <span class="photo-label">場所や日付など</span>
       </div>
   </div>
   ```

### 色や雰囲気の変更
- `style.css` の `:root` セクションにある変数を書き換えることで、サイト全体のテーマカラーを変更できます。

## デザインのこだわり
- **質感**: 背景に微細なノイズを重ね、古いフィルム写真や紙のような質感を再現しています。
- **動線**: スクロールに合わせて要素が静かに浮かび上がるアニメーションを採用しています。
- **ロゴ**: クラシックなバイクエンブレムをイメージしたオリジナルのSVGロゴを搭載しています。
